# 8086 Assembly Lab Programs

This repository contains all my **8086 Microprocessor Lab Programs** from the University of Dhaka.  
Each program is written in **EMU8086 assembly language**, following lab exam-friendly formats.

## ğŸ”¹ Features

- All programs are **lab-exam friendly** and follow the `emu8086.inc` format.
- Programs include **direct user input** using DOS interrupt `int 21h`.
- Single-digit or small multi-digit numbers handled for lab practice.
- Comments added for easy understanding and viva preparation.

## ğŸ”¹ Usage

1. Open EMU8086.
